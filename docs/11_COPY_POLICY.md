# 카피 정책 v0.1

## 1. 제품 한 줄 설명

```text
한국어 문서를 제출하기 전에 문체, AI 작성 리스크, 출처·인용, 제출 안정성을 한 번에 점검하는 서비스입니다.
```

## 2. 랜딩 카피

### Hero title

```text
제출 전, 문서의 리스크를 먼저 확인하세요
```

### Hero subtitle

```text
문체 자연도, AI 작성 리스크, 출처·인용, 제출 안정성을 한 번에 점검하고 문장별 개선 방향을 확인할 수 있습니다.
```

### CTA

```text
문서 점검 시작하기
```

## 3. 기능 카피

| 기능 | 제목 | 설명 |
|---|---|---|
| 문체 자연도 | 문체가 자연스럽게 읽히는지 확인 | 반복 구조, 과도하게 정돈된 문장, 어색한 한국어 표현을 점검합니다. |
| AI 작성 리스크 | AI 작성 리스크 신호 확인 | 작성자를 단정하지 않고, 추가 검토가 필요한 문장 패턴을 보여줍니다. |
| 출처·인용 리스크 | 근거가 필요한 문장 확인 | 주장과 수치, 인용이 필요한 문장에 출처 보완을 안내합니다. |
| 제출 안정성 | 제출 전 체크리스트 제공 | 문서 목적, 길이, 일관성, 수정 여부를 최종 확인할 수 있습니다. |

## 4. 리포트 카피

### 상단 안내

```text
이 결과는 작성자를 단정하지 않습니다.
문체와 문서 패턴을 바탕으로 제출 전 검토가 필요한 신호를 보여주는 참고 지표입니다.
```

### 낮은 리스크

```text
현재 문서는 제출 전 기본 점검 기준에서 큰 위험 신호가 낮습니다.
```

### 보통 리스크

```text
일부 문장에서 문체 일관성 또는 근거 보완이 필요할 수 있습니다.
```

### 높은 리스크

```text
문서 일부 구간에서 추가 검토가 필요한 신호가 발견되었습니다. 문장별 근거를 확인해 주세요.
```

### 짧은 글 안내

```text
짧은 문서는 분석 신뢰도가 낮을 수 있습니다. 결과를 참고 지표로만 활용해 주세요.
```

## 5. 오류 카피

| 상황 | 문구 |
|---|---|
| 빈 입력 | 검사할 문서를 입력해 주세요. |
| 너무 짧음 | 짧은 문서는 분석 신뢰도가 낮을 수 있습니다. |
| 글자 수 초과 | 한 번에 검사할 수 있는 글자 수를 초과했습니다. |
| 분석 실패 | 리포트를 생성하지 못했습니다. 다시 시도해 주세요. |
| 복사 실패 | 제안문을 복사하지 못했습니다. 직접 선택해 복사해 주세요. |

## 6. 토스트 카피

| 상황 | 문구 |
|---|---|
| 복사 성공 | 제안문을 복사했습니다. |
| 저장 성공 | 리포트를 임시 저장했습니다. |
| 다운로드 준비 중 | PDF 다운로드는 다음 버전에서 제공됩니다. |
| 재검사 | 입력 화면으로 돌아갑니다. |
